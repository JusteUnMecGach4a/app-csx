# Application Mobile des Centres Socioculturels de Bar-le-Duc

Bienvenue sur le dépôt de l'application mobile (Progressive Web App) des Centres Socioculturels de Bar-le-Duc (CSC Bar-le-Duc).

## 📱 À propos

Cette application a été conçue pour permettre aux usagers de consulter facilement les horaires, les activités et les actualités des trois centres :
- **Marbot**
- **Libération**
- **Côte Ste-Catherine**

Elle offre une expérience utilisateur fluide et est installable sur smartphone directement depuis le navigateur (technologie PWA).

## 🚀 Déploiement

L'application est déployée automatiquement via GitHub Pages.
Vous pouvez y accéder directement ici :
👉 **[https://justeunmecgach4a.github.io/app-csx/](https://justeunmecgach4a.github.io/app-csx/)**

## 🛠️ Technologies utilisées

- **HTML5 / CSS3 / JavaScript (Vanilla)**
- **Progressive Web App (PWA)** : utilisation d'un `manifest.json` et d'un `service-worker.js` pour permettre l'installation sur mobile.
- Hébergement via **GitHub Pages**.
